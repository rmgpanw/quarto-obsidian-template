# Project Conventions

## Style

- Do not use em dashes (the `—` character). Use regular dashes, commas, or rewrite the sentence instead.

## Structure

- Notes live in `notes/` as `.md` files by default. Use `.qmd` only when executable code chunks are needed.
- The `_template.md` file is the starting point for new notes.
- Wikilinks (`[[note-name]]`) are converted to HTML links at render time by `scripts/wikilinks.lua`.
- The graph visualisation data is generated by `scripts/build-graph.py` during pre-render.

## Build

- Preview: `quarto preview`
- Render: `quarto render`
- Output goes to `_output/` (gitignored).
